# CV-CREATOR
## ReadyGrad

ReadyGrad is an AI-powered career readiness platform that helps fresh graduates prepare for jobs instead of blindly applying to them.

Users upload their skills, experience, and resume once. The platform compares their profile with real job postings and provides:
- 🎯 A readiness score for each job
- 📊 Skill gap analysis
- 📄 AI-powered resume tailoring
- 🌐 Portfolio website generation
- 💡 Plain-language job requirement explanations

Powered by **Gemma 4**, ReadyGrad helps users understand what skills they need to improve and creates a clear roadmap to become job-ready.

For employers, ReadyGrad provides a smarter hiring experience by ranking applicants based on job readiness and helping evaluate candidates more efficiently.

**Our goal:** Replace guesswork in job hunting with personalized guidance and data-driven hiring.
